# Manual Guía para la Gobernanza de Activos Digitales en PYMES

Este proyecto ofrece una plantilla adaptable para implementar gobernanza digital en pequeñas y medianas empresas, basada en cuatro dominios (Alineación, Riesgo, Rendimiento, Comunicaciones), cuatro funciones (Supervisión, Control, Integración, Toma de decisiones) y cuatro niveles (Organización, Portafolio, Programa, Proyecto).

## Estructura del repositorio
- `/docs`: Contenido principal del manual
- `/templates`: Plantillas por sector
- `/examples`: Casos de uso
- `/assets`: Recursos visuales

## Objetivo
Facilitar la toma de decisiones éticas, estratégicas y sostenibles en la gestión de activos digitales.
